## 使用说明

## Framework
Express 处理请求与发送响应
express-validator 数据验证（有效性与格式）
MongoDB NoSQL数据库存储数据
Mongoose操作MongoDB

## Path
app.js -> 查看挂载的router -> 路由处理函数controller-> 使用Model和Schema存入MongoDB数据 -> express-validator数据验证 -> Auth封装 -> Model层面md5加密
